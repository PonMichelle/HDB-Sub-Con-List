# HDB Subcontractor Directory

Team-shared subcontractor directory with live sync via JSONBin.io.

## Setup (one-time, done by team leader)

1. Go to [jsonbin.io](https://jsonbin.io) and create a free account
2. Copy your **Master Key** from Profile → API Keys
3. Open the app URL → paste the key → click Connect
4. Share the **same key** with all team members via WhatsApp

## Team access

Share your GitHub Pages URL:  
`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

Each team member opens the link, enters the shared API key once, and they're in sync.

## Features

- 890 subcontractors across 120 trades
- Multiple projects — each with independent tracking
- Per-subcontractor: Invitation ✉ / Not Quote ✕ / Addendum 1 & 2 / Quotation received 📥
- Each checkbox auto-stamps today's date
- Card view & table view
- Export to CSV per project
- Live sync every 20 seconds across all team members
